# 📊 AdventureWorks Executive Sales Dashboard

## Project Overview

This project is an interactive Business Intelligence dashboard developed using Microsoft Power BI. It analyzes sales performance, profitability, customer behavior, product performance, and return trends using the AdventureWorks dataset. The dashboard enables business users to monitor KPIs and gain actionable insights through interactive visualizations.

---

## Business Problem

Organizations require a centralized dashboard to monitor key business metrics, identify sales trends, evaluate product performance, and track customer behavior. This dashboard helps decision-makers analyze business performance and make data-driven decisions.
